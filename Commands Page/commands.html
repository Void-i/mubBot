!ruleskip (Song ID WITH FORMAT) <Rule number/Written out rule> - This is probably the most complicated command so I will go into more detail. To ruleskip the currently playing song the format would be !ruleskip <Rule #> you can always replace the rule number with a written out rule, such as 'Removed video'. To add any other song to ruleskip the format would be !ruleskip <song ID> <Rule #>. The song ID MUST include the format. For youtube it would look like this 1:I4aM_S5fyoM for soundcloud it would look like this 2:58ve93cn
